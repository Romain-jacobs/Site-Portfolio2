Site web Portfolio

Ce site a été réalisé pour l'un de nos projets du BUT MMI, j'ai dû réaliser cela seul.
Le sujet du projet était de réaliser un site web Portfolio qui retranscrit tous les projets réalisés au cours du premier semestre en ayant accès à des pdf qui justifient les compétences qu'on a acquéri lors des projets.
J'ai dû utiliser une base de données en JavaScript et la traiter du JavaScript pour pouvoir l'afficher sur le site.
J'ai du aussi réaliser quelques animations CSS comme des hovers pour changer de couleur certains éléments.
Tout cela m'a permis d'augmenter mon niveau dans l'utilisation des langages informatiques, et à réussir à me sortir de situations délicates.
